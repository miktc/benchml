# Grids Submodule

## Contents

The `grids/` submodule contains:

* [`__init__.py`](./__init__.py): Marks this directory as a Python package
* [`dtc_grid.py`](./dtc_grid.py): Renders the parameter grid UI for a `DecisionTreeClassifier` model
* [`dtr_grid.py`](./dtr_grid.py): Renders the parameter grid UI for a `DecisionTreeRegressor` model
* [`gbc_grid.py`](./gbc_grid.py): Renders the parameter grid UI for a `GradientBoostingClassifier` model
* [`gbr_grid.py`](./gbr_grid.py): Renders the parameter grid UI for a `GradientBoostingRegressor` model
* [`knc_grid.py`](./knc_grid.py): Renders the parameter grid UI for a `KNeighborsClassifier` model
* [`knr_grid.py`](./knr_grid.py): Renders the parameter grid UI for a `KNeighborsRegressor` model
* [`linearreg_grid.py`](./linearreg_grid.py): Renders the parameter grid UI for a `LinearRegression` model
* [`linearsvc_grid.py`](./linearsvc_grid.py): Renders the parameter grid UI for a `LinearSVC` model
* [`lr_grid.py`](./lr_grid.py): Renders the parameter grid UI for a `LogisticRegression` model
* [`mlpc_grid.py`](./mlpc_grid.py): Renders the parameter grid UI for a `MLPClassifier` model
* [`mlpr_grid.py`](./mlpr_grid.py): Renders the parameter grid UI for a `MLPRegressor` model
* [`rfc_grid.py`](./rfc_grid.py): Renders the parameter grid UI for a `RandomForestClassifier` model
* [`rfr_grid.py`](./rfr_grid.py): Renders the parameter grid UI for a `RandomForestRegressor` model
* [`ridge_grid.py`](./ridge_grid.py): Renders the parameter grid UI for a `Ridge` model
* [`svr_grid.py`](./svr_grid.py): Renders the parameter grid UI for a `SVR` model
* [`README.md`](./README.md): Overview of the contents within the submodule
