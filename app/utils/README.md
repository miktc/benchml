# Utils Submodule

## Contents

The `utils/` submodule contains:

* [`__init__.py`](./__init__.py): Marks this directory as a Python package
* [`convert.py`](./convert.py): Converts the data type of variables
* [`csv_writer.py`](./csv_writer.py): Creates a CSV file of the dataset
* [`env.py`](./env.py): Detects the runtime environment
* [`json_writer.py`](./json_writer.py): Creates a JSON file of the parameter grid
* [`try_util.py`](./try_util.py): Calls functions inside a try-except block
* [`README.md`](./README.md): Overview of the contents within the submodule
