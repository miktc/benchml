# Core Submodule

## Contents

The `core/` submodule contains:

* [`__init__.py`](./__init__.py): Marks this directory as a Python package
* [`central.py`](./central.py): Class containing Streamlit session state keys
* [`clean_keys.py`](./clean_keys.py): Removes keys from the Streamlit session state
* [`get_param_grid.py`](./get_param_grid.py): Renders the UI for editing the parameter grid
* [`ml_models.py`](./ml_models.py): Trains machine learning models
* [`sample_dataset.py`](./sample_dataset.py): Loads a sample dataset
* [`synthetic_data.py`](./synthetic_data.py): Injects synthetic data into the dataset
* [`visuals.py`](./visuals.py): Creates model evaluation visualizations
* [`README.md`](./README.md): Overview of the contents within the submodule
